files()¶

audb.info.files(name, *, version=None, cache_root=None)[source]¶

Media files included in the database.

Parameters

  • name (str) – name of database

  • version (Optional[str]) – version of database

  • cache_root (Optional[str]) – cache folder where databases are stored. If not set audb.default_cache_root() is used

Return type

list[str]

Returns

media files

Examples

>>> audb.info.files("emodb", version="1.4.1")[:2]
['wav/03a01Fa.wav', 'wav/03a01Nc.wav']
