Database dependencies¶

Media and table files of databases are stored in archive files. A database can also reuse an archive file from a previous version of a database if its content hasn’t changed.

We keep track of those dependencies and store some additional metadata about the audio files like duration and number of channels in a dependency table in a file db.csv for every version of a database.

You request a audb.Dependencies object with:

deps = audb.dependencies('emodb', version='1.1.0')

You can see all entries with

deps()
